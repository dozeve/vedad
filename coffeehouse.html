CoffeeHouse

Landing stranica za lokalni kafić — fokus na rezervacije, meni i lokaciju.

Pogledaj sekcije Rezultati

O projektu

Jednostavan, brz i topao vizual. Primarni CTA: Rezerviši sto. Sekundarni: Pogledaj meni.

  • Minimalan JS, visoki Core Web Vitals
  • Map link za navigaciju (Google/Apple)
  • Hero meni + dnevne ponude

Tech

HTML, CSS (grid/flex), lagani JS; optimizovane slike; a11y i semantika.

Brzina

  • Lazy-load slika
  • Inline kritični CSS
  • Keširanje na CDN-u

Ključne sekcije

Meni

Kafe, čajevi, kolači — jasan cjenovnik i istaknute preporuke.

Lokacija

Link ka mapi i radno vrijeme. CTA “Dođi danas”.

Brza forma (EmailJS) i potvrda na email.
